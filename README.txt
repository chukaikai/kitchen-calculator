Kitchen OS v0.4 更新方式

1. 解壓縮本資料夾。
2. 將以下 5 個檔案拖入原本 GitHub/kitchen-calculator 資料夾：
   index.html
   weekly.html
   daily.html
   manifest.webmanifest
   sw.js
3. 遇到同名檔案，選擇「取代」。
4. GitHub Desktop：Summary 輸入 Kitchen OS v0.4。
5. 按 Commit to main，再按 Push origin。
6. 等 1～2 分鐘後開啟原網址並重新整理。

重要：
- 週盤／回推計算機會載入 GitHub 歷史版本中的原始完整檔案，包含原配方、換算與過敏原資料。
- 每日盤點資料保存在使用者當下裝置的瀏覽器 localStorage。
- 不要刪除 GitHub Repository，也不要清除瀏覽器網站資料。
